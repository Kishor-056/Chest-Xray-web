# 🚀 Quick Start Guide

## Step 1: Install Dependencies
```bash
npm install
```

## Step 2: Start the Application
```bash
npm run dev
```

## Step 3: Open in Browser
Navigate to: `http://localhost:3000`

## That's it! 🎉

---

## ✅ Checklist
- [ ] Backend API is running on `http://localhost:8000`
- [ ] npm install completed successfully
- [ ] npm run dev started the server
- [ ] Browser opened at localhost:3000
- [ ] You see the dashboard

---

## 🎯 First Steps

1. **Check Dashboard** - Verify system status is "Online"
2. **Go to Single Prediction** - Click "Single Prediction" in sidebar
3. **Upload Image** - Click upload area and select a chest X-ray
4. **Select Model** - Choose from 6 available models
5. **Click Predict** - Get your AI analysis!

## ⚙️ Configuration

**Change API URL:**
1. Go to Settings (bottom of sidebar)
2. Enter new API URL
3. Click "Save Settings"

**Change Default Model:**
1. Go to Settings
2. Select model from dropdown
3. Click "Apply Model Switch"

---

## 🆘 Common Issues

**"Backend not responding"**
- Start your backend server first
- Check it's on `http://localhost:8000`

**"npm install fails"**
- Delete `node_modules` folder
- Delete `package-lock.json`
- Run `npm install` again

**"Port 3000 already in use"**
- Stop other applications on port 3000
- Or change port in `vite.config.js`
